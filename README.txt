
LOGO PACKAGE

Arquivos gerados:
- assets/images/logo-text.png      (principal, fundo transparente)
- assets/images/logo-text@2x.png   (retina)
- assets/images/logo-text@4x.png   (extra alta resolução)
- assets/images/logo-text.webp     (opção mais leve)
- assets/images/favicon-32.png, favicon-48.png, ... favicon-128.png
- assets/images/og-image.png       (para compartilhamento em WhatsApp/Facebook)
- assets/css/overrides.css         (troca a logo no header sem mudar HTML)

Como aplicar (sem mexer no HTML):
1) Copie os PNG/WebP para a pasta do projeto: assets/images/
2) Substitua seu assets/css/overrides.css por este novo (ele já troca a logo automaticamente)
3) (Opcional, SEO) No <head> do index.html, adicione favicons e og-image.
